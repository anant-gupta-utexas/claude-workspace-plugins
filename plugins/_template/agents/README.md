# Agents

Place agent definition files (.md) in this directory.

## Agent File Structure

Each agent file should be a markdown file with descriptive content about the agent's role, expertise, and behavior.

Example:
```markdown
# Expert Python Developer

You are an expert Python developer specializing in...

## Skills
- Python 3.10+
- FastAPI
- pytest

## Guidelines
- Follow PEP 8
- Write type hints
- Include docstrings
```

## Naming Convention

Use descriptive names like:
- `python-expert.md`
- `data-scientist.md`
- `api-architect.md`
